# Earth-Scientist
Earth Scientist is a 2D platform game developed using the Phaser 3 library. The project is part of a study on sustainability and environmental protection, where the player controls a scientist on a critical mission: to collect improperly discarded waste in a native forest to restore the ecosystem's balance

# 🌍 Eco-Guerreiro: Missão Terra
**Projeto desenvolvido para a Global Game Jam 2026**

Um jogo de aventura e conscientização ambiental onde você controla um cientista em uma missão crítica para salvar a fauna local e limpar os oceanos.

## 🎮 Como Jogar
* **Setas / WASD**: Movimentação e Pulo.
* **Tecla [E]**: Coletar lixo e interagir com animais (Onça, Tartaruga e Golfinho).
* **Espaço**: Passar diálogos.

## 🚀 Tecnologias Utilizadas
* Phaser 3 (Framework de jogos 2D)
* JavaScript / HTML5
* Pixel Art (Estética Retro)

## 🌐 Idiomas Suportados
* Português (PT)
* Inglês (EN)
* Chinês (ZH)
